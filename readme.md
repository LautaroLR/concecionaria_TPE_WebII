# TPE Web II - Concesionaria

**Integrantes:**
- Franco Maidana Escabues - francomaidanaescabues@gmail.com
- Lautaro López Roth - llopezroth@gmail.com

**Temática:** Concesionaria de venta de vehículos usados y 0km.

**Descripción:** Sitio web para una concesionaria donde se pueden visualizar vehículos disponibles. El administrador puede crear, modificar y eliminar categorías y vehículos. Cada vehículo pertenece a una única categoría (relación 1:N).


**Tecnologías:** MySQL, PHP(Server Side Rendering), JavaScript
